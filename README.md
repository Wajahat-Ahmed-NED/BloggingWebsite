### Run Locally

To run **Blogger** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Wajahat/Blogger.git
```

Windows:

```bash
git clone https://github.com/Wajahat/Blogger.git
```
